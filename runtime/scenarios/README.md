# Сохранённые сценарии

Интерфейс сохраняет пользовательские сценарии в эту папку как проверенные JSON-файлы.
Папка и файлы `*.json` не исключаются из Git, поэтому выбранные сценарии можно добавить в коммит и передать вместе с проектом.

JSON содержит параметры FCM, горизонт расчёта и значения шести управляемых индексов.
